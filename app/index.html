Visualizing Baseball Statistics Using Crossfilter.js (back to top)

The charts displayed below result from the multi-dimensional filtering mechanisms that come with Crossfilter.js. The data being visualized comes from Lahman's dataset on how many all-star games certain baseball players have appeared in over the years. Everytime one of the sliders is adjusted, the data is filtered accordingly across all dimensions and the charts are re-rendered and updated to reflect the new subset. As previously mentioned, the main libraries used are Crossfilter.js (Version 1.3.12) and D3.js (Version 3.5.6). More information about Lahman's dataset can be found here.

The Biological Variations of Iris', Examined through DC.js and Crossfilter (back to top)

This visualization is shows the variable species, sepal lengths and widths, and petal lengths and widths of iris flowers. Each chart is linked together, so selecting a subset of one or multiple dimensions will filter and update the other dimensions accordingly. The dataset API used is here. The libraries utilized include:
DC.js, Version: 1.7.5
Underscore.js, Version: 1.8.3
Crossfilter.js, Version: 1.3.12

Customize Your Donut with D3.js (back to top)

This visualization uses the D3.js library to display hierarchical relationships in our Donut dataset. The dataset contains a menu featuring several donut items, each corresponding to its own set of batters, toppings and fillings. We also added in interactive radial transitions to fold and unfold children when a node is clicked. As previously mentioned, the library used for this visualization is D3.js (Version 3.5.6).



The Evolution of Ebola, 2014-2016: A Story in C3.js (back to top)

Ebola took a massive toll on African countries, particularly in recent years. This visualization examines the number of Ebola case from 2014 to present day, how it has evolved, and the severity of the disease in various nations. The charts highlight the staggering numbers and notable trends. The data used was pulled from here. Libraries used include:
D3.js, Version 3.5.6
C3.js, Version 0.4.10
